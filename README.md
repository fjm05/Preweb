# Laboratorio de Análisis Pokémon

Aplicación React para el Pre Parcial Final: consumo de PokéAPI, React Router y estado global con Context API.

## Requisitos cumplidos

- **Rutas:** `/` (listado), `/pokemon/:name` (detalle), `/analysis` (lista de análisis)
- **fetch** con estados de carga, error y mensaje sin datos
- **Estado global** (Context API): agregar/eliminar Pokémon en análisis (máx. 4, sin duplicados)
- **Búsqueda** por nombre en el listado principal
- **Componentes** reutilizables (`PokemonCard`, `Loading`, `ErrorMessage`, `Layout`)

Repositorio: [fjm05/Preweb](https://github.com/fjm05/Preweb.git)

## Ejecutar

Desde la carpeta del repo (`Preweb`):

```bash
npm install
npm run dev
```

Abrir la URL que muestra Vite (por defecto `http://localhost:5173`).

## Build

```bash
npm run build
```
